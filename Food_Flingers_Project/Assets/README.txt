This build contains the Food Flingers game as prepared for Assignment 3.

- 1 Map
- 3 Foods
- Local Multiplayer up to 4 players
- Basic player personalisation

Controls - Menu
WASD/L-Joystick - Navigate
Space/A/✖ - Select
(This game is missing a return button in menus ☠, please reset the build should you want to return to the main menu)

Controls - Game
WASD/L-Joystick - Move around
LMB/X/□ - Grab/Throw food
Space/A/✖ - Dash

Many sounds sourced courtesy of SoundSnap
